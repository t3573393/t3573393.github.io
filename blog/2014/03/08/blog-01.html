获取泛型类中的实际类型

March 8, 2014

fartpig

java中获取泛型中的实际类型方法

public Type getType() {

Type superClass = getClass().getGenericSuperclass();

return ((ParameterizedType) superClass).getActualTypeArguments()[0];
}

例如：

new TypeReference<List<String>>() {}.getType();//结果为  List<String>
