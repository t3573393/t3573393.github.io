pagination-sql 库

April 18, 2019

fartpig

pagination-sql 是一个基于hibernate-core中的dialet中的sql分页查询的实现。项目地址为pagination-sql。

该项目主要是依赖于 hibernate dialect getLimitString 或 LimitHandler.getProcessedSql，对其中的代码进行抽取和转化。 用法为： 
//for oracle >= 9
String sql = "SELECT * FROM ANIMAL";
System.out.println("dataBase: " + Database.ORACLE);
Paginator paginator = new Paginator(Database.ORACLE);
String resultSql = paginator.paginate(sql, 10);
System.out.println("resultSql: " + resultSql);
resultSql = paginator.paginate(sql, 0, 10);
System.out.println("resultSql: " + resultSql);
System.out.println("===================");
